# PRD.md — Product Requirements Document

**محصول**: CSV Cleaner SaaS
**نسخه**: 0.1 (MVP)
**تاریخ**: 2026-05-18

---

## ۱. خلاصه محصول

CSV Cleaner یه ابزار وب است که به استارتاپ‌ها و تیم‌های داده کمک می‌کنه فایل‌های CSV کثیف رو بدون کدنویسی، در چند ثانیه پاک کنند.

---

## ۲. مشکلی که حل می‌کنه

استارتاپ‌ها و تیم‌های کوچک اغلب با CSV هایی کار می‌کنند که:
- ردیف‌های تکراری دارند
- سلول‌های خالی دارند
- نام ستون‌ها ناهماهنگ است (فاصله، بزرگ/کوچک)

راه‌حل فعلی: Excel دستی یا کد Python — هر دو وقت‌بر و خطاپذیر.

---

## ۳. مخاطب هدف

- **Primary**: استارتاپ‌های تکنولوژی که با داده کار می‌کنند
- **Secondary**: فریلنسرها و آنالیست‌های داده
- **Not target**: شرکت‌های بزرگ با data pipeline اختصاصی

---

## ۴. User Story ها

### MVP (v0.1)

```
به عنوان یه کاربر،
می‌خوام فایل CSV ام رو آپلود کنم،
تا نسخه تمیز شده رو دانلود کنم.
```

```
به عنوان یه کاربر،
می‌خوام ببینم چند ردیف تکراری حذف شد،
تا مطمئن بشم داده‌ام درست پردازش شده.
```

### آینده (v0.2+)

```
به عنوان کاربر پولی،
می‌خوام بیشتر از ۵ فایل در ماه آپلود کنم،
تا بتونم روی پروژه‌های بزرگ‌تر هم ازش استفاده کنم.
```

```
به عنوان کاربر،
می‌خوام AI بهم بگه چه مشکلات دیگه‌ای توی داده‌ام هست،
تا بتونم کیفیت داده رو بالاتر ببرم.
```

## ۸. Non-Goals (MVP)

- پشتیبانی از Excel (.xlsx)
- ویرایش داده توی browser
- collaboration چند نفره
- scheduled cleaning
- database connectors

---

## ۹. Technical Constraints

- فایل‌های بزرگ‌تر از 10MB در MVP پشتیبانی نمی‌شوند
- فقط CSV — نه Excel، JSON، یا Parquet
- پردازش synchronous — برای فایل‌های بزرگ ممکن است timeout شود
